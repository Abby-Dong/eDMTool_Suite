How to Use

      
Pick the workflow that fits your goal

      
    

    

      
      

        
Single Component

        
Customize one component → edit the rest in MCE

        
Best when you only need to adjust one block's colors or content before importing, and plan to finish editing inside the MCE editor.

        
    
          
  1. 1
    Browse the Component Library → click a card
    2. 
          
  2. 2
    Adjust colors, inline text, images and links in the Customizer
    3. 
          
  3. 3
    Click Copy HTML
    4. 
          
  4. 4
    In MCE, paste into a Free Form block — all mc:edit regions remain editable
    5. 
        

      

      
      

        
Full eDM

        
Build the full eDM here → export to MCE

        
Best when you want to assemble, customize, and finalize the entire email layout in one place before sending to MCE.

        
    
          
  1. 1
    Click Customize my eDM to open the eDM Builder
    2. 
          
  2. 2
    Drag components from the left panel — drop at the exact position you want
    3. 
          
  3. 3
    Click any block to customize colors; click any text, link or image to edit inline
    4. 
          
  4. 4
    Use Undo / Redo freely; switch the global eDM Theme any time
    5. 
          
  5. 5
    Click Copy Full HTML or Download HTML → paste into MCE Free Form block
    6.
